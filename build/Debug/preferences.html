Preferences

Using the DreamCatcher preferences you can specify the number of link levels you want DreamCatcher to traverse through when checking your Web site, the language and character encoding to use, add and remove words from the dictionary, and specify the script to use when correcting errors.
Maximum Depth to follow links:
Type the maximum number of levels you want DreamCatcher to traverse through when checking the spelling of your Web site. Note that this will not limit the number of pages that DreamCatcher will check, but simply how far from the initial page DreamCatcher will go.
Spellchecker language:
Select a language for the Spellchecker. All of the languages you have installed on your computer for spellchecking will be available in the menu.
Default character encoding:
Select a character encoding to use when spellchecking your Web site. The default character encoding is ISO-8859-1. When DreamCatcher checks a site, it first checks with the Web server to find out whether a character encoding header is specified. If not, then it uses the character encoding set here in the Preferences pane. DreamCatcher does not listen to character encoding directives embedding in HTML documents.
DreamCatcher dictionary:
Use the plus and minus buttons to add and remove words from the DreamCatcher dictionary. Modifications to the dictionary will be saved between sessions. For more information about modifying the dictionary from the DreamCatcher results list refer to Modifying the Dictionary.
Script Well:
Drag custom scripts or the ErrorHandler.scpt file that came with DreamCatcher into the Script Well. The scripts dictate how DreamCatcher fixes misspellings. Refer to Correcting Errors for more information about setting up the script and correcting misspellings.
When you are done making modifications to the preferences click Save Changes.
