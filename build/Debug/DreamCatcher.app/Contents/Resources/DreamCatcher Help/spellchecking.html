Spell Checking

Type the web address of a Web site beginning with http:// into the search field. Then click Begin Spellchecking.
To Spell Check using DreamCatcher:
  1. Type the Web address of the Web site you want to check into the DreamCatcher text field.
  2. Click Begin Spellchecking.
  3. Watch as DreamCatcher traverses your Web site and checks each page.
  4. When the search is complete, the results are displayed as a list.
The results are grouped by the pages that contain the misspelled words. To view the spelling errors on a specific page click on the link to that page. All of the misspelled words are listed and a web view of the page is shown below. Select a word to see it highlighted in the web view. If DreamCatcher has any suggestions for correcting the error they are listed below each word. DreamCatcher uses the Apple system spell checker to search for spelling errors and provide correction suggestions. Refer to correcting errors and ignoring errors for more information.
You can open as many DreamCatcher windows as you like by selecting File > New. You can also export your results to an HTML file by selecting File > Export Results to HTML... This command generates an HTML file that you can use to send the list of spelling errors to others who are not using DreamCatcher.
Note: DreamCatcher does not support frames and will not be able to spellcheck framesets. You can use the NOFRAMES tag to provide links to your framed content that DreamCatcher (and other user agents that do not support frames) can follow.
