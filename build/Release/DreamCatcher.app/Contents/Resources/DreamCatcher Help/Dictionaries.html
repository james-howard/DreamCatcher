Custom Dictionaries

You can add and remove words from DreamCatcher's dictionary by selecting the word and selecting Addor Remove from the Spelling menu. To view a list of words you have added to the dictionary, select Preferences from the DreamCatcher menu. When you add a word to the dictionary it will remain in the dictionary everytime you use DreamCatcher. The default lanuguage for DreamCatcher is English, however, you can easily change the language in the preferences pane. Select Preferences from the DreamCatcher menu. In the Preferences pane you can choose from a variety of languages to use when spell checking your Web site.
You can open as many DreamCatcher windows as you like by selecting File > New. You can also export your results to an HTML file by selecting File > Export Results to HTML... This command generates an HTML file that you can use to send the list of spelling errors to others who are not using DreamCatcher.
